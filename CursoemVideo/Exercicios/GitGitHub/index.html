Git e GitHub

Vamos saber as diferenças de Git e GitHub!

Repositório local/remoto

O Repositório é um lugar ao qual você guarda coisas.

Existem dois tipos de repositórios:

O repositório que será guardado em seu computador (local)

E o repositório que será online e também fazer backup dos arquivos (remoto)

Git

O Git pode ser utilizado para fazer backups de seus sites, por exemplo para que quando faça alterações no site, possa voltar para a versão anterior ao qual não teve modificações.

GitHub

O GitHub pode ser utilizado para fazer backups fora do computador, isto é, utilizando-se de um servidor para armazenar seus arquivos, útil em caso de danos ao armazenamento local da máquina.
