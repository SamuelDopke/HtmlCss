Diferenças entre ID e Class.

ID

IDs deixam o elemento mais único visto que ele só pode nomear um elemento, ou seja, se um id chamado principal foi usado, esse mesmo id principal não pode ser usado em outro elemento.

Class

Class serve para mais de um elemento, ou seja não podemos usar dois h1 diferentes com o mesmo id, mas usando class podemos

Um elemento também pode possuir mais de duas class ou seja <h1 class="destaque somatorio">

As classes destaque e somatorio funcionariam perfeitamente.
