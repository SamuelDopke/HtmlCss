Paleta de Cores

Modo de cores no Adobe color, Paletton e o Coolors

CMYK

  • C = Ciano
  • M = Magenta
  • Y = Amarelo
  • K = Preto

Usadas para cor nas impressoras.

RGB

  • R = Red
  • G = Green
  • B = Blue

Normalmente usaremos este modo de cor

Sites para buscar ou criar sua Paleta:

Adobe Color ótima ferramenta para criar uma paleta de cores com vários recursos, inclusive pegando paletas de uma imagem ou indo atrás de imagens para buscar alguma inspiração para criar uma paleta de cores.

Paletton é também uma ótima ferramenta para montar a paleta de cores, isto porque esse site traz um recurso muito interessante: O exemplo de site que aquela paleta de cores pode criar, isso facilita muito a visualização de como seria o site segundo aquela paleta.

O Coolors é um ótimo site também para achar uma paleta de cores, isso porque ele gera paletas de cores para você aleatóriamente, mas conta com recursos como travar uma cor específica para ela continuar ali e o algoritmo começar a procurar cores que combinam com essa cor travada.
