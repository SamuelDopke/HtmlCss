Imagens dinâmicas

Como foi feita a dinâmica das imagens:

        
<picture>
    <source media="(max-width: max largura)" srcset="localda/imagem.png" type="imagem/pngOUjpg"> // Comece sempre pela mais pequena para não haver erros
    <source media="(max-width: 1050px)" srcset="imagens/media.png" type="image/png"> // Exemplo correto

</picture>
