Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> e </b> (não semântica)

Nesta frase temos um termo em destaque usando a tag <strong> e </strong> (sêmantica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> e </i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> e </em> (semântica)

Texto Marcado

Podemos criar também um Texto Marcado usando a tag <mark> e </mark>

Aqui criei outro Texto Marcado para mostrar que há como mudar a cor da marcação usando CSS, na própria página HTML.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo. Sendo texto grande entre as tags já obsoletas(Não mais recomendadas a utilização) <big> e </big>. E o texto pequeno entre as tags que são permitidas e recomendadas <small> e </small>

Texto Deletado

Podemos marcar um texto deletado como excluído que deve ser lido mas não considerado

Texto Inserido

Podemos marcar um texto inserido para dar uma ênfase e mostrar que ele foi adicionado depois

Texto Sobrescrito

Para inserir coisas do tipo x3 + 4, o 3 para ficar sobrescrito foi utilizado a tag <sup> e </sup>

Texto Subscrito

Para inserir coisas do tipo H20, e para deixar o 2 subscrito foi utilizado a tag <sub> e </sub>
