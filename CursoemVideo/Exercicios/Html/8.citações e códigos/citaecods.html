Código-fonte / Pré-formatação

O comando: document.getElementById('teste') é escrito em linguagem JavaScript e para ficar com a letra mono-espaçada foi utilizado a tag <code> e </code>

            
<pre>
    <code>
        var c = 1 + 5 
        var a = 3 + 2 
        var b = c + a 
        document.write(`${b}`)
    </code>
</pre>

Para deixar o código dessa forma, organizado, foi usada a tag antes do <code> e </code>, a tag, <pre> e </pre>, como no exemplo acima.

Citações simples

Há uma frase de Dostoiévski que me lembro bem, e que levo comigo no meu âmago: A verdade é sempre inverosímil. Para colocar como citação usamos a tag <q> e </q>

Citações completas

O livro Memórias do Subsolo de Dostoiévski se inicia assim:

Sou um homem doente... Sou um homem malvado. Sou um homem desagradável. Creio que tenho uma doença do fígado. Aliás, não compreendo absolutamente nada da minha moléstia e não sei mesmo exatamente onde está o mal.
Para fazer essa citação completa usamos a tag <blockquote> e </blockquote>

Abreviações

Estou estudando HTML e CSS. Estou adorando! Para usar essas abreviações com sinalização para o que significa e essa marcação em conjunto, usamos a tag <abbr> e </abbr>
